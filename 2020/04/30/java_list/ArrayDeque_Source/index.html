总结

ArrayDeque 队列， 队列的特性就是对数据是先进先出。 而栈的特性是先进后出(比如枪打出去的子弹).

这里的理解就是, 使用二个变量，然后每次进行 add / get / remove 都是利用这二个变量来进行 添加 / 删除 / 获取等操作.
